File	Description
Pdam_specific.txt	Contains one row for each Pdam-specific ortholog group/gene family. 'ORTHOMCLXXX' is the gene family name. Followed by number of genes in family, and names of genes (corresponding to fasta headers in PocilloporaDamicornis.pep)
Pdam_specific_all.fasta	Fasta amino acid sequences for all Pdam genes in all Pdam-specific gene families.  
pdam_goterms_counts.txt	All GO terms associated with all pdam genes in all pdam-specific gene families. Column 1=GO term, Column 2=number of time GO term occurred among all pdam genes in all pdam-specific gene families.
